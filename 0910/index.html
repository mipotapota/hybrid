📋 프로젝트 개요

🛠️ 사용 기술

Android Studio Java Kotlin XML SQLite Git

🎯 주요 구현 기능

  • 사용자 인터페이스 설계 및 구현
  • 데이터베이스 연동 및 데이터 관리
  • 액티비티 간 데이터 전달
  • 리스트뷰 및 어댑터 패턴 구현
  • 이벤트 처리 및 사용자 상호작용

📚 학습 성과

이 프로젝트를 통해 Android 애플리케이션 개발의 전체 과정을 경험했습니다. 특히 사용자 중심의 UI/UX 설계와 효율적인 데이터 관리 방법을 학습할 수 있었습니다.
